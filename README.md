Microbe Cataolgue
=================

A webapp for Microbe's data catalogue

Tech stack:

- Node
- React
- [React Admin](https://marmelab.com/react-admin)
- Vite

## installation 

```
# install node 18, if you don't have it
nvm install 18
# install app
npm install
```

## startup
```bash
npm run dev
```
